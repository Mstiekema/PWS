Interactieve en Dynamische Website

Welkom

Voor een website gebruik je meer dan een progameertaal. Zo heb je html, css en javascript. Deze verschillende talen hebben allemaal hun eigen rol en functie binnenin een website. Zo html voor wat de website toont, css voor de styling en javascript voor de functies achter de website.

Een banaan is een langwerpige, licht gebogen vrucht, die afkomstig is van de bananenplant. In Suriname spreekt men van bacove als het om de vrucht gaat die rechtstreeks uit de schil gegeten kan worden en van banaan als het om groene bakbananen gaat. In Indische gerechten wordt de banaan meestal pisang genoemd, het Maleise woord voor de vrucht. Ook in het Afrikaans kent men dat woord, maar dan gespeld als piesang.

Een van de 'applicaties' van onze website is een versimpelde versie van paint. Onze versie maakt gebruikt van een 32x32, een 24x24 en een 16x16 groot oppervlak. De grootte van dit oppervlak hangt af van de grootte van je scherm. Verder kan je gebruik maken van 8 kleuren om je mooie creaties op het doek te toveren.

Er zijn veel manieren om een website te bouwen. Voor ons PWS hebben we besloten om de server code te schrijven in de taal Node.js. Dit komt omdat het redelijk eenvoudig is en steeds vaker wordt gebruikt voor het bouwen voor website.

Achter een website zit vaak ook een database. In een database wordt bepaalde informatie opgeslagen die weer kan worden opgehaald. In een database kan bijvoorbeeld login informatie opgeslagen worden. Er zijn verschillende soorten databases die je kan gebruiken en ieder heeft zo zijn eigen voordelen. Wij zelf gebruiken MySQL omdat deze het meest bekend is en veel mogelijkheden biedt.

Een systeem om in te loggen op een website is een mooi voorbeeld van samenwerking tussen de website en de server erachter.
