Zoeken

Op de website kan je ook zoeken naar dingen. Je kan naar random woorden zoeken of je zoekt naar bepaalde keywords om op andere bestaande pagina's te komen. Hieronder kan je zien welke keywords je naar welke pagina brengt.

  • Basics | HTML, CSS, JavaScript, JS, basics, programmeertalen, programmeertaal, taal, talen
  • Uren | uren, logboek, tijd
  • Paint | paint, kleuren, kleur
  • Login | login, account, nieuw account
  • Website uitleg | website, backend, nodejs, server
  • Database uitleg | database, mysql, tables
  • Login uitleg | login uitleg
  • Paint uitleg | paint uitleg, kleur uitleg
Om dit te doen, hebben we natuurlijk wat JavaScript nodig. Eerst maken we een object waar we alle keywords per pagina invoeren als array. Daarna loopen we door het object kijken we of het gezochte woord in de array staat van de bepaalde pagina. Als dat waar is, dan sturen we de gebruiker naar die pagina en stoppen we de rest van de zoek functie.

Als je naar andere woorden zoekt komt je op deze pagina terecht en kan je bovenaan zien naar welk woord je hebt gezocht. Dit gebeurt door eerst de waarde van de zoekbox input te halen en dan door te sturen naar /zoek/:woord. Als je naar iets probeert te zoeken zonder iets ingevoerd te hebben, krijg je een error melding dat je eerst wat moet opzoeken, omdat je anders een niet bestaande pagina zou krijgen.
