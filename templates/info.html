Welke talen gebruikt deze website en wat doen ze?

HTML basics

HTML is als eerste niet een programmeertaal, het is een "Markup Language". Daarmee wil er gezegd worden dat je met HTML de indeling van een webpagina bepaald, ver niets. Dit zijn vooral hele simpele dingen, maar staan wel aan de fundering van je website.

Titels / koppen
In HTML kan je heel makkelijk titels aangeven om een nieuw stuk tekst duidelijk te maken. Dit kan door middel met de "heading tags". Deze tags kunnen van grootte 1 tot en met 6 gaan (h1 t/m h6).

Zo ziet H1 eruit, de grootste header

Zo ziet H6 eruit, de kleinste header

Deze headers worden veel gebruikt aan de bovenkant van pagina's om te laten zien hoe de pagina heet. Bovendien kan je zien dat de font van de headers in 'Times New Roman' staan, dit is de standaard font van html. Om dit te veranderen gebruik je css, wat later aan bod komt.

Links
Als je op DEZE link klikt, dan opent de home page in dit tabblad. Als je echter op DEZE link klikt, dan opent de home page op een ander tabblad.

CSS basics

CSS is, net zoals HTML, een markup language. Met CSS kan je echter veel meer doen qua het ontwerpen van het uiterlijk van je website. Zo kan je bijvoorbeeld tekst een kleur geven of spelen met hoe het font eruit ziet. Dit word gedaan door middel van het aanpassen van bestaande HTML codes: het desbetreffende gedeelte van de HTML code wat je wilt veranderen geef je een id of class. Naar dit id of deze kun je referren in je CSS bestand en de gewilde aanpassingen aanbrengen. Dit kan er dan als volgt uitzien: 

.balk {
      text-align: center;
      border-top: 1px solid #216983;
      border-bottom: 2px solid #216983;
      margin-bottom: 5px;
    }

In het voorbeeld word voor alles met het id balk de tekst gecentreerd en worden er een border en margin ingesteld, waar je iets meer hieronder over kunt lezen.

Belangrijk op een website is de positie van elemten op zichzelf en tegenover een ander. Dit word ook geregeld met CSS, voornamelijk met de begrippen Position, margin, border en padding. JavaScript basics

Met JavaScript kan je de pagina op vele manieren beïnvloeden. Hierdoor kun je een webpagina interactief maken voor de gebruiker. Zo kan je bepaalde "events" waarnemen zoals een klik op het beeldscherm of als je over een bepaald item heen beweegt met je muis. Je script kan hiermee dan functies activeren, praten met de server of direct dingen veranderen op de pagina zelf
