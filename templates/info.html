HTML basics

HTML is als eerste niet een programmeertaal, het is een "Markup Language". Hiermee wordt bedoelt dat je met HTML kan bepalen hoe de pagina eruit ziet qua indeling. Dit zijn vooral hele simpele dingen.

Titels / koppen
In HTML kan je heel makkelijk titels aangeven om een nieuw stuk tekst duidelijk te maken. Dit kan door middel met de "heading tags". Deze tags kunnen van grootte 1 tot en met 6 gaan (h1 t/m h6).

Zo ziet H1 eruit, de grootste header

Zo ziet H6 eruit, de kleinste header

Deze headers worden veel gebruikt aan de bovenkant van pagina's om te laten zien hoe de pagina heet.

Links
Als je op DEZE link klikt, dan opent de home page in dit tabblad. Als je echter op DEZE link klikt, dan opent de home page op een ander tabblad.

CSS basics

CSS is, net zoals HTML, een markup language. Met CSS kan je echter veel meer doen qua het ontwerpen van het uiterlijk van je website. Zo kan je bijvoorbeeld tekst een kleur geven of spelen met hoe het font eruit ziet.
